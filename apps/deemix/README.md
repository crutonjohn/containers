# deemix

## Custom environment configuration

This container support setting certain custom enviroment variables.

| Name                            | Default             |
|---------------------------------|---------------------|
| DEEMIX__DATA_DIR                | `/config`           |
| DEEMIX__MUSIC_DIR               | `/downloads`        |
| DEEMIX__HOST                    | `0.0.0.0`           |
| DEEMIX__PORT                    | `6595`              |
